POST /report/:uuid Report image

Report an image with a specific uuid as abusive.

Request

/report/:uuid { reason: 'explain briefly why this picture is inappropriate' }

Example

/report/48780a63-df1f-48d1-99ee-6c60e535a274 { reason: 'picture contains nudity' }

Response

Status: 200 OK

GET /validate Validate random image

This method returns a randomly chosen image together with some usuful metadata.

Example

/validate

Response

Status: 200 OK { label: 'dog' provider: 'donation' url: '/donations/48750a63-df1f-48d1-99ee-6c60e535a271' uuid: '48750a63-df1f-48d1-99ee-6c60e535a271' }

GET /validate/:uuid Validate specific image

This method returns a specific image together with some usuful metadata.

Request

/validate/:uuid

Example

/validate/48750a63-df1f-48d1-99ee-6c60e535a271

Response

Status: 200 OK { label: 'dog' provider: 'donation' url: '/donations/48750a63-df1f-48d1-99ee-6c60e535a271' uuid: '48750a63-df1f-48d1-99ee-6c60e535a271' }

GET /export Export datasets

This method returns a collection of images that belong to one or more specific tags.

Example

/export?tags=dog

Response

Status: 200 OK { label: 'dog' provider: 'donation' url: '/donations/48750a63-df1f-48d1-99ee-6c60e535a271' uuid: '48750a63-df1f-48d1-99ee-6c60e535a271' probability: 0.9230769 num_yes: 12 num_no: 1 }
